# 주식 현황 & 뉴스 → 텔레그램 전송

## 역할
너는 주식 시황 분석가야. 아래 지침을 순서대로 빠짐없이 실행해.

---

## Step 1: 현재 시각 확인
```bash
date +"%Y-%m-%d %H:%M KST" --date="TZ=Asia/Seoul now" 2>/dev/null || TZ=Asia/Seoul date +"%Y-%m-%d %H:%M KST"
```

---

## Step 2: 국내 증시 시세 수집

아래 스크립트를 실행해:
```bash
python3 /home/user/stock-bot/fetch_krx.py
```

실패 시 WebSearch로 대체: `코스피 코스닥 현재가 site:finance.naver.com`

---

## Step 3: 해외 증시 시세 수집

아래 스크립트를 실행해:
```bash
python3 /home/user/stock-bot/fetch_global.py
```

실패 시 WebSearch로 대체: `나스닥 S&P500 현재 지수`

---

## Step 4: 24시간 이내 핵심 뉴스 5건 수집

WebSearch를 사용하되 **반드시 지난 24시간 필터 적용**:

1. `주식 증시 코스피 뉴스` (지난 24시간)
2. `나스닥 미국증시 뉴스` (지난 24시간)
3. `반도체 AI 주식 뉴스` (지난 24시간)

각 뉴스 확인:
- 발행 시각이 현재 기준 24시간 이내인가?
- URL에서 날짜 확인 가능한가? (불가 시 제외)

뉴스 5건을 아래 형식으로 정리:
```
[N] 제목: {제목}
    시각: {YYYY-MM-DD HH:MM KST}
    요약: {100자 이내 핵심 요약}
    감성: {호재/중립/악재}
    링크: {URL}
```

---

## Step 5: 텔레그램 메시지 작성 및 전송

아래 형식으로 메시지를 작성해. 총 길이 3800자 이내.

```
📊 주식 & 뉴스 시황
⏰ {현재날짜} {현재시각} KST

━━━━ 🇰🇷 국내 증시 ━━━━
코스피  : {지수} {▲/▼} {등락폭} ({등락률}%)
코스닥  : {지수} {▲/▼} {등락폭} ({등락률}%)
코스피200: {지수} {▲/▼} {등락폭} ({등락률}%)

━━━━ 🌏 해외 증시 ━━━━
나스닥      : {지수} {▲/▼} {등락폭} ({등락률}%)
S&P500      : {지수} {▲/▼} {등락폭} ({등락률}%)
다우존스    : {지수} {▲/▼} {등락폭} ({등락률}%)
필라반도체  : {지수} {▲/▼} {등락폭} ({등락률}%)

━━━━ 📰 오늘의 핵심 뉴스 ━━━━
[1] {제목}
    {시각} | {감성}
    {요약}
    🔗 {링크}

[2] ~ [5] 동일 형식

🤖 Claude 자동 시황 알림
```

메시지 작성 후 아래 명령어로 전송:
```bash
python3 /home/user/stock-bot/send_telegram.py "{메시지 전체}"
```

전송 실패 시: 메시지 전체를 채팅창에 출력해줘.
