오늘 날짜와 현재 시각을 KST 기준으로 먼저 확인해줘.
bash 명령어: date +"%Y-%m-%d %H:%M KST"

# 📊 STEP 1 — 주식 현황 수집

WebSearch를 사용해서 아래 종목들의 **현재가 / 전일 대비 등락폭 / 등락률**을 수집해줘.
검색어 예시: "삼성전자 주가 현재" / "TSLA stock price today" 등을 활용해.

## 국내 증시 (KRX)
| 종목 | 코드 |
|------|------|
| 삼성전자 | 005930 |
| SK하이닉스 | 000660 |
| 코스피 지수 | — |
| 코스닥 지수 | — |

## 해외 증시
| 종목 | 티커 | 비고 |
|------|------|------|
| 마이크로소프트 | MSFT | NASDAQ |
| 테슬라 | TSLA | NASDAQ |
| 알파벳(구글) | GOOGL | NASDAQ |
| 메타 | META | NASDAQ |
| 스페이스X | — | 비상장 → 뉴스만 |
| 나스닥 지수 | ^IXIC | — |
| S&P 500 | ^GSPC | — |

수집한 데이터를 아래 형식으로 정리해줘:

```
【 🇰🇷 국내 증시 】
삼성전자   : {현재가}원  {▲/▼} {등락폭}원 ({등락률}%)
SK하이닉스 : {현재가}원  {▲/▼} {등락폭}원 ({등락률}%)
코스피     : {지수}  {▲/▼} {등락폭} ({등락률}%)
코스닥     : {지수}  {▲/▼} {등락폭} ({등락률}%)

【 🌏 해외 증시 】
마이크로소프트(MSFT) : ${현재가}  {▲/▼} {등락폭} ({등락률}%)
테슬라(TSLA)        : ${현재가}  {▲/▼} {등락폭} ({등락률}%)
알파벳(GOOGL)       : ${현재가}  {▲/▼} {등락폭} ({등락률}%)
메타(META)          : ${현재가}  {▲/▼} {등락폭} ({등락률}%)
나스닥              : {지수}  {▲/▼} {등락폭} ({등락률}%)
S&P 500             : {지수}  {▲/▼} {등락폭} ({등락률}%)
```

데이터를 확인할 수 없는 종목은 "확인불가" 라고 표시해줘.

# 📰 STEP 2 — 오늘의 핵심 뉴스 (24시간 이내만)

오늘이 며칠인지 먼저 확인해줘.
그리고 **현재 시각으로부터 정확히 24시간 이내에 발행된 뉴스만** 다뤄야 한다. 24시간을 초과한 기사는 단 한 건도 포함하지 마.
오늘 한국 투자자가 알아야 할 핵심 경제 뉴스를 정리해줘. 목표는 5건이다.

## 시간 규칙 (가장 중요)
- 기준 시각: 이 작업이 실행되는 현재 시각 (KST)
- 허용 범위: 현재 시각 - 24시간 이내에 발행된 기사만
- 검색 시 Google Search의 시간 범위 필터를 "지난 24시간"으로 활용해줘
- 검색 결과 URL이나 메타데이터에서 발행 날짜를 확인하고, 24시간을 초과한 경우 제외
- 발행 시각을 확인할 수 없는 기사는 제외

## 데이터 원칙
- Google Search Grounding을 반드시 사용해줘
- 검색으로 확인되지 않는 수치·인용은 출력하지 마
- 페이월 본문은 인용하지 마. 공개 보도만 사용해
- 출처 URL이 없는 문장은 출력하지 마

## 신뢰 가능한 매체 (한 곳에서만 보도되어도 통과)
- **한국 경제·금융 매체**: 연합뉴스·뉴스1·뉴시스·한국경제·매일경제·조선비즈·이데일리·머니투데이·서울경제·파이낸셜뉴스·헤럴드경제·아시아경제·인포맥스·비즈워치·머니S·SBS Biz·한국경제TV·MTN·에너지경제 등 경제 전문 매체
- **글로벌 매체 (무료 헤드라인)**: Reuters·Bloomberg·WSJ·FT·CNBC·MarketWatch·Yahoo Finance·Nikkei·SCMP
- **공식 출처**: 한국은행·금감원·기획재정부·미 연준·ECB·BOJ 등 중앙은행 발표, 정부 보도자료, DART/SEC 공시
- ❌ 블로그·개인 채널·광고성 PR·티스토리·네이버 카페·유튜브 영상은 제외

## 선별 기준 (둘 중 하나라도 충족)
**A. 한국 투자자 자산에 직간접 영향**
다음 산업·자산에 영향이 있는 뉴스:
- KOSPI/KOSDAQ 시총 상위 종목
- 한국 주요 산업: 반도체·2차전지·조선·방산·바이오·자동차·철강·화학·정유·금융(은행·증권·보험)·인터넷·통신·엔터·게임·유통·건설
- 원/달러 환율, 한국 금리, 한국 부동산 정책
- **모니터링 종목 관련**: 삼성전자·SK하이닉스·마이크로소프트·테슬라·스페이스X·구글(Alphabet)·메타

**B. 한국 시장에 곧 반영될 글로벌 뉴스**
한국 시장 시간대(개장 전~장중~마감 후)에 발표되어 당일 한국 시장이 반응할 가능성이 있는 글로벌 이슈:
- 미 연준·ECB·BOJ 통화정책
- 미 빅테크·반도체 실적 및 가이던스
- 원자재(유가·금·구리), 미중·중동 지정학

## 중복 제거 (필수)
- 같은 사건/이슈를 다른 언론사가 보도한 기사는 반드시 1건만 선택
- 제목이 달라도 핵심 내용(사건·수치·주체)이 같으면 중복으로 처리

## 단계별 분석 (각 뉴스마다 내부적으로 수행 후 출력)

**Step 1. 발행 시각 확인**
- 검색 결과에서 기사 발행 시각을 추출 (YYYY-MM-DD HH:MM KST 형식)
- 24시간을 초과한 경우 즉시 제외하고 다른 뉴스 검색

**Step 2. 사실 확인**
- 제목만 보지 말고 본문 검색 결과까지 확인해서 핵심 사건·수치·주체를 추출

**Step 3. 요약 (200자 이내)**
- 핵심 사건·수치·주체·영향 요인을 200자 이내로 정리

**Step 4. 감성 분석 (150자 이내)**
- 호재 / 중립 / 악재 중 하나로 분류
- 확정 표현(계약 체결·실적 발표·정책 시행)은 강하게 반영
- 모호 표현(검토 중·논의 중·전망)은 중립 처리
- 반복 보도된 이슈는 감성 강도 약화
- 업종별 민감도 고려 (제약: 임상, 반도체: 고객사 이탈 등)
- "매수하세요·매도 신호" 같은 지시어 절대 금지

**Step 5. 리스크 분석 (100자 이내)**
- 7가지 리스크 유형 중 해당 여부 판단:
  ① 규제 ② 평판 ③ 지배구조 ④ 기술/제품 ⑤ 재무 ⑥ 소송 ⑦ 사업환경
- 시점: 단기 / 중기 / 구조적
- 리스크 없으면 "없음 - (간단한 사유)"로 명시

## 출력 전 자기검증 (반드시 수행)
출력하기 전에 각 뉴스에 대해 다음을 스스로 체크해줘:
1. 발행 시각이 현재로부터 24시간 이내인가? → 초과 시 제외
2. 발행 시각이 명확한가? → 불확실하면 다시 검색해서 재확인
3. 출처 URL이 실재하는가? → 없으면 다시 검색
4. **만약 후보가 5건 미만이라면, 산업·종목·매체 범위를 확장해서 한 번 더 검색해줘.** 24시간 이내 한국 시장 관련 의미있는 뉴스는 정상적인 평일이면 5건 이상 존재한다.

## 오답 방지 규칙
- 제목만으로 판단하지 마
- MOU는 계약이 아님 → 호재로 단정 금지
- 소규모 소송은 악재 과잉 해석 금지
- 자극어("폭락"·"충격"·"대란") 사용 금지

## 출력 형식 (반드시 아래 양식을 정확히 따를 것)

🌅 오늘의 핵심 뉴스 (YYYY-MM-DD KST)

[1] 📌제목: {{기사 제목}}
⊙ 발행시간: {{YYYY-MM-DD HH:MM KST}}
⊙ 요약: {{200자 이내 핵심 요약}}
⊙ 감성 분석: {{호재 / 중립 / 악재}} - {{150자 이내 근거}}
⊙ 리스크 분석: {{있음 / 없음}} - {{유형·시점·영향 요약, 100자 이내}}
⊙ 링크: {{기사 URL}}

[2] 📌제목: ...
(이하 동일)

## 출력 건수
- **목표: 5건**
- 5건이 안 나오면 검색 범위를 한 번 더 확장해서 재시도해줘
- 단, 페이월·발행시각 불명·URL 부재 기사를 억지로 채우는 것은 금지

Take a deep breath and work through this step by step. 시간 규칙을 가장 엄격하게 지키고, 5건 목표를 달성하도록 검색을 충분히 시도해줘.

# 📲 STEP 3 — 카카오톡 전송

위 STEP 1(주식 현황)과 STEP 2(뉴스 5건)를 합쳐서 하나의 메시지로 만든 뒤 카카오톡으로 전송해줘.

메시지 구조:
```
📊 주식 & 뉴스 시황
⏰ {현재날짜} {현재시각} KST

━━━━ 🇰🇷 국내 증시 ━━━━
{국내 종목 현황}

━━━━ 🌏 해외 증시 ━━━━
{해외 종목 현황}

━━━━ 📰 오늘의 핵심 뉴스 ━━━━
{뉴스 5건 요약 (각 뉴스당 제목 + 발행시각 + 요약 2줄 + 링크)}

🤖 Claude 자동 시황 알림
```

전송 명령어:
```bash
cd /home/claude/stock_reporter && \
  export KAKAO_ACCESS_TOKEN=$(grep KAKAO_ACCESS_TOKEN .env 2>/dev/null | cut -d= -f2) && \
  python3 send_kakao.py "위에서 만든 메시지 전체"
```

전송 실패 시: 메시지 전체를 채팅창에 그대로 출력해줘.
